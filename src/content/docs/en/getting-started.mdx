The Grovery’s CORE <InlineGlossary text="RMS" />&nbsp;(Resource Management System) is a software product that provides the framework to **organize**, **manage** and **train** Sales and Marketing teams. This <InlineGlossary text="Web App platform" />&nbsp;provides the environment for teams to work more efficiently, share information and provides easy access to relevant program materials.
